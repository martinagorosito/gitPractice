La pericia documentológica nos permite el análisis de todo tipo de documentos, papel moneda, escritos o firmas, diferentes tipos de impresiones; para detectar si los mismo son fidedignos o falsificados.

Esto se realiza a través de una yuctaposición entre un documento dubitado y un indubitado, comparándolos para detectar las falsificaciones (si las hay).
